    ```python
    def _extract_ac_power(inverter_registers: list[int]) -> float | None:
        """Extract AC power (W) from Model 103 inverter registers.

        Model 103 layout: DID(0) + Len(1) + 50 data fields.
        AC Power at data offset 14 (register index 16), scale factor at data offset 15 (register index 17).
        Values are unsigned 16-bit. Scale factor is signed 16-bit (two's complement).
        """
        if len(inverter_registers) < 18:
            return None
        raw_power = inverter_registers[16]
        raw_sf = inverter_registers[17]
        if raw_power == 0xFFFF or raw_sf == 0x8000:
            return None  # Not implemented / invalid
        # Convert unsigned to signed for scale factor
        sf = raw_sf if raw_sf < 0x8000 else raw_sf - 0x10000
        return float(raw_power) * (10.0 ** sf)
    ```

    Look at aggregation.py to verify the register offsets match the existing AC power decode pattern. The SunSpec Model 103 W field is at offset 14 from the model data start, which is index 16 in the full register list (index 0 = DID, index 1 = length, then 50 data fields starting at index 2).
