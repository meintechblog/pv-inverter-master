1. Define `SHELLY_SERVICE_TYPE = "_shelly._tcp.local."` constant.
2. Implement `async def discover_shelly_devices(timeout: float = 3.0, skip_ips: set[str] | None = None) -> list[dict]`:
   - Use `AsyncZeroconf`, `AsyncServiceBrowser` with `on_state_change` callback (identical to mqtt pattern).
   - After browser cancel, resolve each found name via `AsyncServiceInfo`.
   - Extract IP from `info.addresses[0]` via `socket.inet_ntoa()`, fallback to `info.server.rstrip(".")`.
   - Parse TXT record properties (bytes to str): extract `gen` (default "1"), `app` (model name), `ver` (firmware).
   - Build result dict: `{"host": ip, "name": service_name, "generation": f"gen{gen}", "model": app, "firmware": ver}`.
   - Filter out any host in `skip_ips` set (per D-08 dedup pattern from pitfall 4 in research).
   - Log with `structlog`: `log.info("shelly_mdns_scan_complete", devices_found=len(results), timeout=timeout)`.
   - Always `await aiozc.async_close()` in finally block.

3. Implement `async def probe_shelly_device(host: str, timeout: float = 5.0) -> dict`:
   - Use `aiohttp.ClientSession` with `aiohttp.ClientTimeout(total=timeout)` to GET `http://{host}/shelly`.
   - Parse JSON response. Determine generation: `gen_value = data.get("gen", 0)`, gen2+ if `gen_value >= 2` else gen1.
   - Extract model: `data.get("app", data.get("type", "Unknown"))` (Gen2 uses "app", Gen1 uses "type").
   - Extract mac: `data.get("mac", "")`.
   - Return `{"success": True, "generation": "gen1"|"gen2", "model": model, "mac": mac, "gen_display": f"Gen{gen_value}" if gen_value >= 2 else "Gen1"}`.
   - On any exception: return `{"success": False, "error": f"Could not reach Shelly at {host}: {e}"}`.
   - Use 5-second timeout per research pitfall 2.

```python
async def shelly_probe_handler(request: web.Request) -> web.Response:
    """POST /api/shelly/probe -- probe a Shelly device at given host."""
    try:
        body = await request.json()
        host = body["host"]
    except (KeyError, ValueError, TypeError) as e:
        return web.json_response({"error": f"Invalid request: {e}"}, status=400)
    result = await probe_shelly_device(host)
    return web.json_response(result)
```

3. **shelly_discover_handler** (place right after shelly_probe_handler):
```python
async def shelly_discover_handler(request: web.Request) -> web.Response:
    """POST /api/shelly/discover -- scan LAN for Shelly devices via mDNS."""
    try:
        config: Config = request.app["config"]
        existing_ips = {inv.host for inv in config.inverters if inv.type == "shelly"}
        devices = await discover_shelly_devices(timeout=3.0, skip_ips=existing_ips)
        return web.json_response({"success": True, "devices": devices})
    except Exception as e:
        log.error("shelly_discover_failed", error=str(e))
        return web.json_response({"success": False, "error": f"Shelly discovery failed: {e}"}, status=500)
```
