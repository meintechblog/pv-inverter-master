Create `src/venus_os_fronius_proxy/device_registry.py` with:

1. **ManagedDevice dataclass**: entry (InverterEntry), plugin (InverterPlugin), device_state (DeviceState), poll_task (asyncio.Task | None)

2. **DeviceRegistry class**:
   - `__init__(self, app_ctx: AppContext, config: Config, on_poll_success: Callable[[str], Awaitable[None]])` -- stores refs, creates empty `_managed: dict[str, ManagedDevice]`
   - `async start_device(self, device_id: str)` -- find entry in config.inverters by id, skip if not enabled, create plugin via plugin_factory(entry, get_gateway_for_inverter(config, entry)), create DeviceState with DashboardCollector + ConnectionManager(poll_interval from config/gateway), create ManagedDevice, store in _managed, add DeviceState to app_ctx.devices[device_id], start poll task via asyncio.create_task(_device_poll_loop(...))
   - `async stop_device(self, device_id: str)` -- pop from _managed, cancel poll_task + await it (catch CancelledError), call plugin.close() (catch Exception), remove from app_ctx.devices, log "device_stopped"
   - `async start_all(self)` -- iterate config.inverters, start_device for each enabled entry
   - `async stop_all(self)` -- stop_device for all keys in list(_managed.keys())
   - `async enable_device(self, device_id: str)` -- call start_device (it already skips if not enabled, so caller must set entry.enabled=True first)
   - `async disable_device(self, device_id: str)` -- call stop_device
   - `get_active_device_ids(self) -> list[str]` -- return list of device_ids currently in _managed
   - `get_active_count(self) -> int` -- len(_managed)

3. **_device_poll_loop async function** (module-level, not method):
   - Parameters: device_id, plugin, device_state, poll_interval, on_success callback, app_ctx
   - Extract pattern from proxy.py _poll_loop but simplified:
     - While True: if app_ctx.polling_paused -> sleep and continue
     - try: result = await plugin.poll()
     - Increment poll_counter["total"]
     - If result.success: increment poll_counter["success"], conn_mgr.on_poll_success(), store last_poll_data on device_state, await on_success(device_id)
     - If not success: conn_mgr.on_poll_failure()
     - except asyncio.CancelledError: raise  (CRITICAL: must be before except Exception)
     - except Exception: increment total, conn_mgr.on_poll_failure()
     - await asyncio.sleep(conn_mgr.sleep_duration)
   - Do NOT handle night mode injection, common translation, or cache writes -- that's AggregationLayer's job (Plan 02)
   - Do NOT handle DashboardCollector.collect() here -- that will be wired in Plan 02 after aggregation exists

Use structlog for logging with device_id bound context.

   - Create mock InverterPlugin (AsyncMock) that returns PollResult(success=True, common_registers=[0]*67, inverter_registers=[0]*52)
   - Mock plugin_factory to return the mock plugin
   - Use real AppContext and DeviceState
   - Use real ConnectionManager
