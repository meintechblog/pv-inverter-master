       **REGISTER_MODELS constant:** A list of dicts defining the SunSpec register layout for the register viewer. Each entry has:
       - "name": human-readable model name (e.g. "Common (Model 1)")
       - "start": start address (e.g. 40002)

       - Nameplate (Model 120): 40121-40148 (DID, Length, DER Type, W Rating, W Rating SF, VA Rating, VA Rating SF, etc.)
       - Controls (Model 123): 40149-40174 (DID, Length, WMaxLimPct, WMaxLim_Ena, etc.)

       **Route handlers:**

       `async def index_handler(request)`:
       - Load index.html from venus_os_fronius_proxy.static package via importlib.resources
       - Return web.Response(text=html, content_type="text/html")
       - If file not found, return web.Response(text="<h1>Frontend not deployed</h1>", content_type="text/html")

       `async def status_handler(request)`:
       - Read shared_ctx from request.app["shared_ctx"]
       - Return JSON: {"solaredge": conn_mgr.state.value, "venus_os": "active" if server running else "unknown", "reconfiguring": app.get("reconfiguring", False)}

       `async def health_handler(request)`:
       - Read shared_ctx
       - Compute uptime = time.monotonic() - request.app["start_time"]
       - Compute poll_success_rate = (poll_counter["success"] / poll_counter["total"] * 100) if poll_counter["total"] > 0 else 0.0
       - Return JSON: {"uptime_seconds": round(uptime, 1), "poll_success_rate": round(rate, 1), "poll_total": total, "poll_success": success, "cache_stale": cache.is_stale, "last_poll_age": round(time.monotonic() - cache.last_successful_poll, 1) if cache._has_been_updated else null}

       `async def config_get_handler(request)`:
       - Read config from request.app["config"]
       - Return JSON: {"host": config.inverter.host, "port": config.inverter.port, "unit_id": config.inverter.unit_id}

       `async def config_save_handler(request)`:
       - Parse JSON body: {"host": str, "port": int, "unit_id": int}
       - Call validate_inverter_config(host, port, unit_id)
       - If validation fails: return web.json_response({"success": False, "error": error}, status=400)
       - Update config object: config.inverter.host/port/unit_id = new values
       - Call save_config(request.app["config_path"], config)
       - Set request.app["reconfiguring"] = True
       - Call await plugin.reconfigure(host, port, unit_id)
       - Set request.app["reconfiguring"] = False
       - Return JSON: {"success": True}
       - On exception: return 500 with error

       `async def config_test_handler(request)`:
       - Parse JSON body: {"host": str, "port": int, "unit_id": int}
       - Validate with validate_inverter_config
       - Create AsyncModbusTcpClient(host, port=port, timeout=5)
       - Try connect + read_holding_registers(40000, 2, slave=unit_id)
       - Return JSON: {"success": True/False, "error": str or null}
       - Always close test client in finally block

       `async def registers_handler(request)`:
       - Read cache from shared_ctx
       - For each model in REGISTER_MODELS, for each field:
         - Read value(s) from cache.datablock.getValues(field["addr"] + 1, field["size"])
         - For string fields (size > 1 and type == "string"): decode uint16 list to ASCII string
         - For single register fields: return raw uint16 value
       - Return JSON array of model groups with field values included

       **App factory:**

       `async def create_webapp(shared_ctx: dict, config: Config, config_path: str, plugin) -> web.AppRunner`:
       - Create web.Application()
       - Store in app dict: shared_ctx, config, config_path, plugin, start_time=time.monotonic(), reconfiguring=False
       - Add routes: GET /, GET /api/status, GET /api/health, GET /api/config, POST /api/config, POST /api/config/test, GET /api/registers
       - Create AppRunner, call await runner.setup()
       - Return runner
