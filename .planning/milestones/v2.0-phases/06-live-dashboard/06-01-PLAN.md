**webapp.py changes:**

1. Add imports at top: `import weakref`, `import json`, `import time`

2. Add `ws_handler` async function:
   - Create `web.WebSocketResponse(heartbeat=30.0)`, prepare it, add to `request.app["ws_clients"]`
   - Get collector from `request.app["shared_ctx"].get("dashboard_collector")`
   - If collector and collector.last_snapshot: send JSON `{"type": "snapshot", "data": collector.last_snapshot}`
   - If collector: build history dict from collector._buffers. For each buffer key, get_all() samples, downsample with `[::10]` step, convert monotonic timestamps to wall-clock using offset `time.time() - time.monotonic()`. Send JSON `{"type": "history", "data": history}`. Only send ac_power_w buffer for sparkline (Phase 6 only needs this one, but send all for future use).
   - Loop `async for msg in ws`: pass (Phase 7 will handle commands). Break on ERROR.
   - Finally block: `request.app["ws_clients"].discard(ws)`

3. Add `broadcast_to_clients` async function:
   - Accept `app: web.Application` and `snapshot: dict`
   - If no ws_clients or empty, return
   - Serialize once: `payload = json.dumps({"type": "snapshot", "data": snapshot})`
   - Iterate over `set(app["ws_clients"])`, try `await ws.send_str(payload)`, catch `(ConnectionError, RuntimeError, ConnectionResetError)` and discard dead clients

4. In `create_webapp()`: add `app["ws_clients"] = weakref.WeakSet()` and `app.router.add_get("/ws", ws_handler)` after existing routes
